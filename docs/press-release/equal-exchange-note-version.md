## タイトル

デザインの対価を、野菜で受け取っています。——「等価交換報酬制度」の話

## 本文

【画像1: 等価交換のイメージ写真】

こんにちは、デザイン・PRスタジオのBOATshipです。

私たちは、制作の対価をお金以外の「価値」で受け取る「等価交換報酬制度」という仕組みを運用しています。今日はその中身と、実際にやってみて分かったことを紹介します。

### 「予算がないからお願いできない」への違和感

この制度は、ある違和感から始まりました。地方の生産者、スタートアップ、NPO、個人クリエイター。彼らと話すたびに聞いてきたのが「予算がないからお願いできない」という言葉です。プロジェクトの可能性や熱量は本物なのに、お金がないというだけで一緒に動けない。それはもったいない、と思ったのです。

彼らの多くは、お金はなくても届けたい価値を持っています。私たちが持っているのはデザインとPRのスキル。だったら、交換すればいい。

### 仕組みはシンプルです

等価交換報酬制度では、デザイン・PR・制作の対価を、お金以外の価値で受け取ります。スキル、場所、人脈、サービス、モノ——双方が「等価」と合意できるものなら、何でも報酬になります。

「等価」の決め方もシンプルで、市場価格ではなく「この取引に納得できるか」をお互いに確認するだけ。このフラットさが、制度のいちばん大事なところだと思っています。もちろん、通常の金銭でのご依頼と並行して運用しているので、円でのお支払いも歓迎です。

### 実際の交換例をふたつ

【画像2: 交換で受け取った野菜の写真など】

ひとつめ。ある農家さんとは、Webサイト制作の対価として毎月の新鮮な野菜を受け取っています。お金の授受がないぶん「発注者と受注者」ではなく「一緒に価値をつくるパートナー」という関係になり、公開後の改善にもずっと伴走しています。

ふたつめ。あるシェアオフィスとは、ブランディングの対価としてコワーキングスペースの利用権を交換しました。スキルと場所の交換は実はいちばん多いパターンで、お金を介さないからこそ、相手の事業を深く理解する関係が続いています。

### 半年やってみて分かったこと

いいことばかりではありません。「等価かどうか」の合意が難しい場面もありますし、先に制作して後から受け取る交換では信頼関係が前提になります。ただ、この課題を乗り越えたプロジェクトほど、関係が長続きすることも分かってきました。

等価交換で得られる最大の価値は、お金では買えないもの——信頼、関係性、経験、コミュニティ——だと感じています。

### お金以外の対価、ご提案ください

等価交換報酬制度は、お金だけが価値の尺度ではない働き方の実験です。運用で得た知見は、オウンドメディア（https://www.boatship.jp/notes/）で公開し続けます。

「うちにはお金はないけど、これならある」——そんなご提案をお待ちしています。お問い合わせは https://www.boatship.jp/contact/ から、「等価交換でのご依頼」を選んでどうぞ。
