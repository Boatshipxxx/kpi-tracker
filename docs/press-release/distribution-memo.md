# 配信先の選定メモ（人間の配信作業用）

> 配信実務は人間が実施。配信前に本メモの【要確認】を確認してください。

## 1. PR TIMES（第一候補）

- **【要確認】スタートアップチャレンジ（無料配信枠）の適用条件を必ず確認してください。** 設立年数・資本金などの条件は変更されることがあります（BOATshipは2022年4月設立。申込時点の最新の条件で適用可否を判断）。適用外の場合は従量課金プランの単発配信で可。
- 配信カテゴリ案: 「働き方・人事」「デザイン」「地方創生」
- 配信曜日・時間: 火〜木の午前中（10〜11時）が既読されやすい定石。月曜朝と金曜夕方は避ける
- 画像: 【画像1〜3】を差し替えてから入稿（アイキャッチは野菜など「絵になる実物」を推奨。ロゴのみは避ける）

## 2. 併用したい無料・自社チャネル（Owned / Shared）

- 自社オウンドメディア: 配信と同日に note.com 転載版を公開（equal-exchange-note-version.md）
- Instagram（https://www.instagram.com/boatship3com/）: 野菜・交換物の写真とともに要約を投稿
- ニュースレター（サイトの Studio Updates 購読者）への告知

## 3. 直接アプローチ候補（Earned）

制度の「絵になりやすさ」（野菜×デザイン）を考えると、以下の系統のメディア・記者に個別に案内する価値があります（媒体名は例。掲載実績・関係の有無は要確認）:

- **働き方・組織系**: 新しい報酬・働き方の実験として（例: サイボウズ式のような組織文化系メディア）
- **デザイン・クリエイティブ系**: スタジオの取り組みとして
- **地方創生・ローカルエコノミー系**: 地方の生産者との交換事例を切り口に
- **一般ビジネス系**: 「お金以外の報酬」という経済ネタとして

個別アプローチ時は、リリース本文よりも「実例1（野菜）」の写真と1〜2行の説明を先に見せるのが有効です。

## 4. 配信後のフォロー

- 配信当日〜3日: 問い合わせフォームの「等価交換でのご依頼」件数と、GA4で `/notes/equivalent-exchange-compensation/` への流入をチェック
- 掲載・言及があれば `reviews/2026-07.md`（月次レビュー）の流入元・問い合わせ欄に記録
- 反応が良ければ、n01/n04 記事の tuningMemo に「リリース経由の流入」の知見を追記して次の企画へ
