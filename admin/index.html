変更を公開する方法

1 「articles.js をダウンロード」をクリック
2 GitHub の magazine/articles.js を開いて鉛筆アイコン（Edit）をクリック
3 内容をすべて削除し、ダウンロードしたファイルの内容を貼り付け
4 「Commit changes」→ 数分でサイトに反映されます
